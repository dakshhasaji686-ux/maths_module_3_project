# PR-3 Inferential Statistics and Hypothesis Testing

## Project Overview
This project performs inferential statistics and hypothesis testing using a healthcare dataset. The analysis is implemented in Python using Pandas, NumPy, SciPy, and Statsmodels.

## Dataset Information
The dataset contains 100 healthcare records with fields such as:
- Age
- Weight
- BMI
- Blood Pressure
- Smoking Status
- Diabetes
- Cholesterol Level
- Glucose Level
- Age Group

## Tasks Performed
1. Formulated hypotheses
2. Calculated confidence intervals
3. Found critical value and p-value
4. Performed t-test
5. Conducted chi-square test
6. Performed ANOVA test
7. Calculated covariance and correlation
8. Interpreted results

## Technologies Used
- Python
- Pandas
- NumPy
- SciPy
- Statsmodels
- Google Colab / Jupyter Notebook


## Conclusion
The project demonstrates the use of inferential statistics techniques to analyze healthcare data and interpret relationships between variables.
